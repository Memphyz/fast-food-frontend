UM AVISO IMPORTANTE!

Esse projeto é piramente um projeto didático, ou seja, não é possivel realizar pedidos reais e nem informações pessoais serão armazenadas com excessão dos dados de login, ou seja, dados de cartão de crédito (não aconselhável usar dados reais) NÃO serão armazenados no front e nem no backend!

Esse é um projeto que foi criado com o propósito de aprender um pouquinho do backend, mas apesar do projeto ser focado no backend que foi feito em node e em breve tentarei refazer o mesmo em outras linguagens, eu não deixei o front totalmente de lado, então apesar de ter ficado "apresentavel" o mesmo pode conter bugs pois não foi testado 100%, mas acredito que os possiveis bugs não vão interferir na experiência.
